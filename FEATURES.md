## 🎯 Как использовать

### Регистрация и вход
1. Откройте приложение
2. Нажмите "Зарегистрироваться"
3. Введите имя, email и пароль
4. После регистрации вы автоматически войдете

### Создание чата
1. Нажмите кнопку "+" в правом верхнем углу
2. Введите email или имя пользователя
3. Выберите пользователя из результатов поиска
4. Чат будет создан автоматически

### Отправка сообщений

**Текст:**
- Введите текст в поле ввода
- Нажмите Enter или кнопку отправки

**Эмодзи:**
- Нажмите кнопку 😊
- Выберите эмодзи из списка
- Эмодзи добавится в текст

**Файлы и изображения:**
- Нажмите кнопку 📎
- Выберите файл
- Файл отправится автоматически

**Голосовое сообщение:**
- Нажмите кнопку 🎤
- Разрешите доступ к микрофону
- Говорите
- Нажмите "Отправить" или "Отмена"

**Видео кружок:**
- Нажмите кнопку 🎥
- Разрешите доступ к камере
- Нажмите ⏺ для начала записи
- Нажмите ⏹ для остановки
- Нажмите "Отправить"

### Настройки

**Изменение темы:**
1. Нажмите ⚙️ в правом верхнем углу
2. Выберите "Тема оформления"
3. Выберите понравившуюся тему

**Изменение размера шрифта:**
1. Откройте настройки
2. Выберите "Размер шрифта"
3. Выберите нужный размер

**Изменение фона:**
1. Откройте настройки
2. Выберите "Фон чата"
3. Выберите понравившийся фон

**Редактирование профиля:**
1. Откройте настройки
2. Нажмите на карточку профиля
3. Введите новое имя

**Выход:**
1. Откройте настройки
2. Прокрутите вниз
3. Нажмите "Выйти"

### Мобильная версия

**Навигация:**
- Свайп вправо или кнопка "←" для возврата к списку чатов
- Свайп влево для открытия чата

**Оптимизация:**
- Уменьшенные кнопки для удобства
- Адаптивная клавиатура

## 🔧 Технические детали

### Real-time обновления
- Сообщения доставляются мгновенно через WebSocket
- Автоматическое переподключение при потере связи
- Экспоненциальная задержка при переподключении
- Максимум 10 попыток переподключения

### Производительность
- Ленивая загрузка изображений
- Оптимизированный рендеринг сообщений
- Умный автоскролл (только если внизу)
- Периодическое обновление чатов (каждые 30 сек)
- Обновление при возврате на вкладку

### Хранение данных
- JWT токен в LocalStorage
- Настройки в LocalStorage
- Сообщения в памяти (загружаются при открытии чата)
- Файлы на сервере

### Обработка ошибок
- Уведомления об ошибках
- Автоматический retry для сетевых запросов
- Graceful degradation при отсутствии функций
- Логирование в консоль для отладки

## 📱 Поддержка браузеров

### Полная поддержка
- ✅ Chrome 90+
- ✅ Firefox 88+
- ✅ Safari 14+
- ✅ Edge 90+

### Частичная поддержка
- ⚠️ Safari < 14 (нет WebRTC для видео)
- ⚠️ IE 11 (не поддерживается)

### Требуемые API
- WebSocket
- LocalStorage
- MediaRecorder (для голоса/видео)
- getUserMedia (для камеры/микрофона)
- Fetch API
- ES6+ JavaScript

## 🐛 Известные ограничения

1. **Видео кружки** работают только в Chrome/Edge (WebRTC)
2. **Голосовые сообщения** не показывают длительность
3. **Файлы** ограничены 100MB
4. **WebSocket** переподключение может занять до 30 секунд
5. **Уведомления** работают только внутри приложения

## 💡 Советы по использованию

1. **Для лучшей производительности:**
   - Закрывайте неиспользуемые чаты
   - Очищайте кэш браузера периодически
   - Используйте современный браузер

2. **Для экономии трафика:**
   - Отключите автозагрузку медиа в настройках
   - Используйте текстовые сообщения

3. **Для приватности:**
   - Выходите из системы на общих устройствах
   - Не сохраняйте пароль в браузере
   - Используйте сложные пароли

4. **Для удобства:**
   - Используйте Enter для отправки
   - Добавляйте часто используемые эмодзи
   - Настройте тему под свои предпочтения
