## 🚀 Как запустить

### Локально
```bash
cp .env.example .env
docker-compose up -d
```
Откройте http://localhost

### На VDS
```bash
cd /opt
git clone your_repo messenger
cd messenger
chmod +x deploy.sh
./deploy.sh
```
Откройте http://your_server_ip

## 💪 Сложности и решения

### Проблема: WebSocket переподключение
**Решение:** Экспоненциальная задержка с максимумом попыток

### Проблема: Адаптивность на мобильных
**Решение:** Media queries + flexbox + тестирование на реальных устройствах

### Проблема: Real-time обновления
**Решение:** WebSocket + оптимистичный UI + проверка дубликатов

### Проблема: Загрузка файлов
**Решение:** Multer + проверка размера + типов файлов
