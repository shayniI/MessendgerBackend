## 🔍 Проверка в DevTools

### Инспектор элементов
1. Откройте DevTools (F12)
2. Выберите инструмент "Выбрать элемент" (Ctrl+Shift+C)
3. Кликните на длинное сообщение
4. Проверьте стили:

```css
.message-text {
    word-wrap: break-word;
    word-break: break-word;
    overflow-wrap: break-word;
    white-space: pre-wrap;
    max-width: 100%;
}

.message-content {
    max-width: 60%; /* или 75%, 85% на мобильных */
    overflow-x: hidden;
}
```

### Проверка overflow
1. В DevTools найдите элемент `.messages`
2. Проверьте что:
   - `overflow-x: hidden`
   - `overflow-y: auto`
3. Прокрутите чат вниз
4. Не должно быть горизонтального скролла

## ❌ Возможные проблемы

### Проблема: Чат все еще растягивается

**Причина:** Кеш браузера

**Решение:**
1. Очистите кеш (Ctrl+Shift+Delete)
2. Перезагрузите страницу (Ctrl+F5)
3. Проверьте что styles.css загрузился

### Проблема: Текст не переносится

**Причина:** Старая версия CSS

**Решение:**
1. Проверьте в DevTools что стили применились
2. Перезапустите Docker: `docker-compose restart nginx`
3. Очистите кеш браузера

### Проблема: Слова разбиваются неправильно

**Причина:** Браузер не поддерживает `overflow-wrap: anywhere`

**Решение:**
1. Обновите браузер до последней версии
2. Используйте Chrome или Firefox
3. Проверьте что используется `word-break: break-word`

### Проблема: На мобильных текст слишком мелкий

**Причина:** Неправильный размер шрифта

**Решение:**
1. Откройте настройки
2. Выберите "Размер шрифта"
3. Выберите "Большой" или "Очень большой"
