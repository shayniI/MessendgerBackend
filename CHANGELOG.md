### Backend API
- POST /api/auth/register - Регистрация
- POST /api/auth/login - Вход
- GET /api/users/me - Профиль
- PUT /api/users/me - Обновление профиля
- GET /api/users/search - Поиск пользователей
- GET /api/users/contacts - Список контактов
- POST /api/users/contacts - Добавить контакт
- POST /api/chats - Создать чат
- GET /api/chats - Список чатов
- GET /api/chats/:id/members - Участники чата
- POST /api/chats/:id/members - Добавить участника
- DELETE /api/chats/:id/members/me - Покинуть чат
- POST /api/messages - Отправить сообщение
- GET /api/messages/chat/:id - Получить сообщения
- PUT /api/messages/:id - Редактировать сообщение
- DELETE /api/messages/:id - Удалить сообщение
- POST /api/messages/:id/read - Отметить прочитанным

### WebSocket Events
- new_message - Новое сообщение
- user_status - Изменение статуса
- typing - Индикатор печати

### Технологии
**Backend:**
- Node.js 18
- Express 4
- WebSocket (ws)
- PostgreSQL 15
- JWT аутентификация
- bcrypt для паролей
- Multer для загрузки файлов

**Frontend:**
- Vanilla JavaScript
- HTML5 + CSS3
- WebSocket API
- MediaRecorder API
- LocalStorage

**Infrastructure:**
- Docker + Docker Compose
- Nginx (reverse proxy)
- Multi-container architecture

### Безопасность
- JWT токены с истечением
- bcrypt хеширование паролей
- Параметризованные SQL запросы
- CORS защита
- Валидация входных данных

### Производительность
- Индексы в PostgreSQL
- Оптимизированные запросы
- Кеширование статики в Nginx
- Сжатие ответов

## Известные проблемы

### v1.0.0
- WebSocket переподключение может занять до 3 секунд
- Голосовые сообщения не показывают длительность
- Видео кружки работают только в Chrome/Edge
- Нет индикатора загрузки файлов
- Нет предпросмотра файлов перед отправкой
